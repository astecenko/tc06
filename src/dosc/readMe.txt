Плагин менеджера структуры, проводит проверку сборки в менеджере структуры на соответствие следующим критериям:
1. Наличие серии внедрения на статусе объекта
2. Заполнение всех граф RVZ СП, кроме графы "кол-во на изд"
3. Наличие материала в объекта типа "Деталь", "Геометрический материал", "Стандартное изделие"
4. Правильность наименования добавленных стандартных изделий (проверка по идентификатору "9_...")
5. Наличие номера извещения на карточке информации об изделии в ревизии обекта (Master Item Revision) - не ошибка, преудпреждение
